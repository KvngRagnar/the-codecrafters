## Go Syntax

    Syntax generally refers to the specific rules and arrangement of words and phrases to create well-formed sentences in a language.

    so in Go, this is the arrangement of words and integers to write a functional code.

    A go file usually consists of a package declaration statement, import packages, functions and statements and expressions.

    A typical Go code looks like:

    package main // this program belongs to the main package
    import ("fmt") // allows import of files in the fmt package
                 // blanks lins are called white, they enhance readability of a code.
    func main () { // this is a function and codes within the curly braces are executed
        fmt.Println("Hello World!")
    }



# Go stateents
    The print command in the code above is a statement. In Go, statements are separated by ending a line using the Enter key which adds a semicolon.
