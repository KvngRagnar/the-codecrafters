# GO STRUCT 

    A struct in Go is a user-defined data type that allows you to group multiple variables of different data types into a single unit. Unlike arrays, which store values of the same type, structs can hold a mix of data types, making them useful for representing real-world entities like records.

    Structs are declared using the type and struct keywords, with fields defined inside curly braces. Each field has a name and a specific data type.

    To access or modify struct members, the dot (.) operator is used with the struct variable. This allows you to read or assign values to each field individually.

    Structs can also be passed as arguments to functions, enabling you to work with grouped data efficiently. Overall, structs help organize related data and make programs more structured and easier to manage.
