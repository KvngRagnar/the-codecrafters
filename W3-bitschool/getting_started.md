# GETTING STARTED

    To start using Go, you need a text editor to write code and a compiler to convert it into a language the computer understands. An IDE like Visual Studio Code is commonly used because it combines both functions in one tool. Go can be downloaded from its official website, and you can confirm installation by running the go version command in the terminal.

    After installing Go, you set up your IDE by installing the Go extension and required tools. Then, you initialize a project using the command go mod init, which prepares your workspace. A simple Go program can be created in a file like helloworld.go, where the main function prints “Hello World!” using the fmt package.

    The program is executed with the go run command, and it can also be compiled into an executable using go build. Overall, the process introduces the basic steps needed to install, set up, and run a Go program successfully.
